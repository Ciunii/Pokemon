#  ポケモン タイプ相性計算ツール

## 📖 使い方

1. **⚔️ こうげきのタイプ**: パネルをクリックして、こうげきの技タイプ（最大4つ）を選択します。
2. **🛡️ ぼうぎょのタイプ**: パネルをクリックして、こうげきの技を受けるタイプを選択します。
3. **📊 結果の確認**: 
    - **ダメージ倍率**: 選択した攻撃技が相手に何倍で入るかを表示します。
    - **おすすめタイプ**: 有効な攻撃タイプを表示します。
    - **シミュレーション**: 複合タイプを含む現171パターンから、特定の条件に合う組み合わせを自動抽出します。
